# Linx 命令行学习备忘
+ cd - 回到上一次的目录，cd 不加参数，就可以直接回到主目录 。
+ file：查看文件的类型，Linux 文件的类型不是由后缀名决定的。  
+ mv：移动文件，更改文件的名字。  
+ less  
+ man： 查看手册
